# wyf9's Blog

个人 blog 站点仓库, 使用 [fuwari](https://github.com/saicaca/fuwari) 构建

Site: [wyf9.top](https://wyf9.top)

## 常用命令

- 启动开发服务器 (`:4321`)

```sh
pnpm run dev
```

- 启动预览服务器

```sh
pnpm run preview
```

- 构建静态站点

```sh
pnpm run build
```
