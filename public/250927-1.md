# GESP 五级：常用 C++ 标准库函数

在 GESP 五级考试中，标准库函数是编程题和选择题的核心。以下是高频函数，按库分类，包含用途、输入输出、示例和考试提示。建议手写代码验证，尤其是 `sort` 和 `vector` 操作。所有函数需 `#include <bits/stdc++.h>` 和 `using namespace std;`。

## 1. 算法库 `<algorithm>`

用于排序、查找和序列操作，编程题常考。

### sort
- **用途**：对数组/vector 排序，默认升序，可自定义比较。
- **输入输出**：`sort(first, last, [cmp])`  
  - `first, last`：排序范围 [first, last)（如 `v.begin(), v.end()`）。  
  - `cmp`：可选比较函数，返回 bool（true 表示 a 排在 b 前）。  
  - 复杂度：O(n log n)。
- **示例**：
  ```cpp
  vector<int> a = {3, 1, 4};
  sort(a.begin(), a.end()); // 升序: {1, 3, 4}
  sort(a.begin(), a.end(), greater<int>()); // 降序: {4, 3, 1}
  ```
- **注意**：需 `<algorithm>`。自定义 cmp（如 `bool cmp(int a, int b) { return a > b; }`）用于降序或结构体排序。考试高频，如贪心算法（区间调度）。

### lower_bound
- **用途**：在有序序列中找第一个 ≥ val 的位置。
- **输入输出**：`lower_bound(first, last, val, [cmp])`  
  - 返回迭代器，需减去 first 得下标。  
  - 复杂度：O(log n)。
- **示例**：
  ```cpp
  vector<int> a = {1, 3, 3, 5};
  auto it = lower_bound(a.begin(), a.end(), 3);
  cout << it - a.begin(); // 1
  ```
- **注意**：序列需升序。常用于二分查找问题。

### upper_bound
- **用途**：在有序序列中找第一个 > val 的位置。
- **输入输出**：同 `lower_bound`。
- **示例**：
  ```cpp
  vector<int> a = {1, 3, 3, 5};
  auto it = upper_bound(a.begin(), a.end(), 3);
  cout << it - a.begin(); // 3
  ```
- **注意**：序列需升序，注意边界（val 不在序列中）。

### reverse
- **用途**：反转序列。
- **输入输出**：`reverse(first, last)`  
  - 反转 [first, last)。  
  - 复杂度：O(n)。
- **示例**：
  ```cpp
  vector<int> a = {1, 2, 3};
  reverse(a.begin(), a.end()); // {3, 2, 1}
  ```
- **注意**：常用于高精度运算（反转结果）。

### max_element
- **用途**：找序列最大值。
- **输入输出**：`max_element(first, last, [cmp])`  
  - 返回最大值迭代器，解引用得值。  
  - 复杂度：O(n)。
- **示例**：
  ```cpp
  vector<int> a = {1, 4, 3};
  cout << *max_element(a.begin(), a.end()); // 4
  ```
- **注意**：`min_element` 同理，考试选择题常考复杂度。

### swap
- **用途**：交换两元素。
- **输入输出**：`swap(a, b)`  
  - 交换变量 a 和 b。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  int x = 1, y = 2;
  swap(x, y); // x=2, y=1
  ```
- **注意**：常用于快速排序实现。

**考试提示**：
- `sort` 是编程题核心（如区间调度），熟练掌握自定义比较（见之前讲解）。
- `lower_bound`/`upper_bound` 用于二分查找，注意序列必须有序。
- 选择题常考复杂度：`sort` O(n log n)，`max_element` O(n)。

## 2. 动态数组 `<vector>`

动态数组，替代固定数组，防越界。

### push_back
- **用途**：尾部添加元素。
- **输入输出**：`v.push_back(x)`  
  - 添加 x 到 vector 尾部。  
  - 复杂度：O(1) 均摊。
- **示例**：
  ```cpp
  vector<int> v;
  v.push_back(1); // v = {1}
  ```
- **注意**：自动扩容，高精度运算常用。

### pop_back
- **用途**：删除尾部元素。
- **输入输出**：`v.pop_back()`  
  - 移除尾部元素。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  vector<int> v = {1, 2};
  v.pop_back(); // v = {1}
  ```
- **注意**：空 vector 调用未定义，需判空。

### size
- **用途**：返回元素个数。
- **输入输出**：`v.size()`  
  - 返回 unsigned int。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  vector<int> v = {1, 2};
  cout << v.size(); // 2
  ```
- **注意**：常用于循环：`for(int i=0; i<v.size(); i++)`。

### empty
- **用途**：判断是否为空。
- **输入输出**：`v.empty()`  
  - 返回 bool。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  vector<int> v;
  cout << v.empty(); // true
  ```
- **注意**：循环前检查空状态。

### clear
- **用途**：清空 vector。
- **输入输出**：`v.clear()`  
  - 移除所有元素。  
  - 复杂度：O(n)。
- **示例**：
  ```cpp
  vector<int> v = {1, 2};
  v.clear(); // v = {}
  ```
- **注意**：不释放内存，考试少用。

### insert
- **用途**：插入元素。
- **输入输出**：`v.insert(it, x)`  
  - 在迭代器 it 前插入 x。  
  - 复杂度：O(n)。
- **示例**：
  ```cpp
  vector<int> v = {1, 3};
  v.insert(v.begin()+1, 2); // v = {1, 2, 3}
  ```
- **注意**：插入慢，考试慎用。

### erase
- **用途**：删除元素。
- **输入输出**：`v.erase(it)` 或 `v.erase(start, end)`  
  - 删除指定位置或范围。  
  - 复杂度：O(n)。
- **示例**：
  ```cpp
  vector<int> v = {1, 2, 3};
  v.erase(v.begin()+1); // v = {1, 3}
  ```
- **注意**：迭代器失效，需重新计算。

**考试提示**：
- vector 常用于动态存储（如高精度运算）。
- 配合 `sort`/`lower_bound` 解决排序/查找问题。
- `size()` 返回 unsigned，循环小心（如 `i >= 0` 可能死循环）。

## 3. 字符串 `<string>`

用于输入处理或字符操作。

### length / size
- **用途**：返回字符串长度。
- **输入输出**：`s.length()` 或 `s.size()`  
  - 返回 unsigned int。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  string s = "abc";
  cout << s.length(); // 3
  ```
- **注意**：与 vector size 类似。

### substr
- **用途**：提取子串。
- **输入输出**：`s.substr(pos, len)`  
  - 从 pos 起提取 len 个字符。  
  - 复杂度：O(len)。
- **示例**：
  ```cpp
  string s = "hello";
  cout << s.substr(1, 3); // ell
  ```
- **注意**：pos 越界抛异常，考试慎用。

### find
- **用途**：查找子串。
- **输入输出**：`s.find(t, pos)`  
  - 从 pos 起找 t，返回首次出现下标或 `string::npos`。  
  - 复杂度：O(n*m)。
- **示例**：
  ```cpp
  string s = "hello";
  cout << s.find("ll"); // 2
  cout << (s.find("x") == string::npos); // true
  ```
- **注意**：`npos` 是 unsigned long long，用 `== string::npos` 判断。

### c_str
- **用途**：转 C 风格字符串。
- **输入输出**：`s.c_str()`  
  - 返回 const char*。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  string s = "hello";
  printf("%s\n", s.c_str()); // hello
  ```
- **注意**：用于 `printf`，考试高频。

**考试提示**：
- 字符串常用于高精度输入或反转。
- `c_str()` 配合 `printf` 避免 cin/cout 超时。

## 4. 数学 `<cmath>`

用于简单数学计算。

### sqrt
- **用途**：开平方。
- **输入输出**：`sqrt(x)`  
  - 返回 double，x 的平方根。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  cout << sqrt(16.0); // 4.0
  ```
- **注意**：输入需 double，x ≥ 0。常用于素数判断。

### pow
- **用途**：幂运算。
- **输入输出**：`pow(base, exp)`  
  - 返回 base^exp (double)。  
  - 复杂度：O(log exp)。
- **示例**：
  ```cpp
  cout << pow(2.0, 3.0); // 8.0
  ```
- **注意**：精度可能丢失，考试少用。

### abs
- **用途**：绝对值。
- **输入输出**：`abs(x)`  
  - 返回 int/long long 的绝对值。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  cout << abs(-5); // 5
  ```
- **注意**：浮点用 `fabs` (<cmath>)，需 `<cstdlib>`。

### ceil / floor
- **用途**：向上/向下取整。
- **输入输出**：`ceil(x)` / `floor(x)`  
  - 返回 double。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  cout << ceil(3.2); // 4.0
  cout << floor(3.7); // 3.0
  ```
- **注意**：结果为 double，需强转 int。

**考试提示**：
- `sqrt` 常用于素数判断（检查到 sqrt(n)）。
- 数据范围大，用 long long 或 double。

## 5. 内存操作 `<cstring>`

用于数组/字符串初始化。

### memset
- **用途**：初始化数组。
- **输入输出**：`memset(arr, val, size)`  
  - 将 arr 前 size 字节设为 val。  
  - 复杂度：O(n)。
- **示例**：
  ```cpp
  int a[100];
  memset(a, 0, sizeof(a)); // 全 0
  memset(a, -1, sizeof(a)); // 全 -1
  ```
- **注意**：val 按字节填充，int 数组只用 0/-1。常用于筛法/DP 初始化。

### memcpy
- **用途**：复制数组。
- **输入输出**：`memcpy(dest, src, size)`  
  - 从 src 复制 size 字节到 dest。  
  - 复杂度：O(n)。
- **示例**：
  ```cpp
  int a[3] = {1, 2, 3}, b[3];
  memcpy(b, a, sizeof(a)); // b = {1, 2, 3}
  ```
- **注意**：注意内存重叠，考试少用。

**考试提示**：
- `memset` 高频，用于数组初始化（如线性筛）。
- `memcpy` 偶尔用于 struct 数组复制。

## 6. 队列 `<queue>`

用于 BFS 或贪心，考试少见。

### push
- **用途**：入队。
- **输入输出**：`q.push(x)`  
  - x 入队尾。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  queue<int> q;
  q.push(1); // q = {1}
  ```
- **注意**：普通队列先进先出。

### pop
- **用途**：出队。
- **输入输出**：`q.pop()`  
  - 移除队首。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  q.pop(); // q = {}
  ```
- **注意**：空队列调用未定义，需 `!q.empty()`。

### front
- **用途**：访问队首。
- **输入输出**：`q.front()`  
  - 返回队首元素。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  cout << q.front(); // 1
  ```
- **注意**：配合 `empty()` 使用。

### priority_queue
- **用途**：优先队列（默认大顶堆）。
- **输入输出**：`priority_queue<int>` 或 `priority_queue<int, vector<int>, greater<int>>` (小顶堆)  
  - push/pop/top 同 queue，复杂度 O(log n)。
- **示例**：
  ```cpp
  priority_queue<int> pq;
  pq.push(3); pq.push(1);
  cout << pq.top(); // 3
  ```
- **注意**：小顶堆需 `greater<int>`。

**考试提示**：
- 优先队列用于贪心（如最小代价）。
- 判空：`while(!q.empty())`。

## 7. 栈 `<stack>`

用于递归替代或括号匹配，考试少见。

### push
- **用途**：入栈。
- **输入输出**：`s.push(x)`  
  - x 压栈。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  stack<int> s;
  s.push(1); // s = {1}
  ```
- **注意**：后进先出。

### pop
- **用途**：出栈。
- **输入输出**：`s.pop()`  
  - 移除栈顶。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  s.pop(); // s = {}
  ```
- **注意**：空栈调用未定义。

### top
- **用途**：访问栈顶。
- **输入输出**：`s.top()`  
  - 返回栈顶元素。  
  - 复杂度：O(1)。
- **示例**：
  ```cpp
  cout << s.top(); // 1
  ```
- **注意**：配合 `empty()` 使用。

**考试提示**：
- 栈可用于括号匹配或递归模拟。
- 判空：`while(!s.empty())`。

## 综合示例

以下代码结合多种库函数，模拟考试场景：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    // vector 排序
    vector<int> a = {3, 1, 4, 1, 5};
    sort(a.begin(), a.end()); // {1, 1, 3, 4, 5}
    cout << *max_element(a.begin(), a.end()) << endl; // 5

    // 二分查找
    cout << lower_bound(a.begin(), a.end(), 3) - a.begin() << endl; // 2

    // 字符串处理
    string s = "hello";
    reverse(s.begin(), s.end()); // olleh
    printf("%s\n", s.c_str());

    // 数组初始化
    int arr[100];
    memset(arr, 0, sizeof(arr));
    cout << arr[0] << endl; // 0

    // 优先队列
    priority_queue<int, vector<int>, greater<int>> pq;
    pq.push(3); pq.push(1);
    cout << pq.top() << endl; // 1
    return 0;
}
```
